Guide a systematic review process following PRISMA 2020 guidelines.

User input: $ARGUMENTS

## Instructions

### Phase 1: Protocol
1. **Define the review question** (PICO)
2. **Specify eligibility criteria**:
   - Study designs to include/exclude
   - Date range, language restrictions
   - Publication status (published, preprint, grey literature)
3. **Plan search strategy**:
   - Databases: PubMed, Embase, Cochrane CENTRAL, Web of Science (minimum)
   - Use PubMed MCP to develop and test the search strategy
   - Document all search strings for reproducibility
4. **Register protocol** (recommend PROSPERO registration)

### Phase 2: Search & Screening
5. **Execute searches** across all databases via PubMed MCP
6. **De-duplicate** results
7. **Screen by title/abstract** — define clear screening criteria
8. **Screen by full text** — document exclusion reasons
9. **Generate PRISMA flow diagram** data:
   - Records identified, duplicates removed
   - Screened, excluded at title/abstract
   - Full-text assessed, excluded with reasons
   - Studies included in qualitative/quantitative synthesis

### Phase 3: Data Extraction
10. **Design extraction form**:
    - Study characteristics (author, year, country, design, N)
    - Population characteristics
    - Intervention/comparison details
    - Outcomes (effect measure, point estimate, CI, p-value)
    - Risk of bias assessment
11. **Pilot the form** on 2-3 studies

### Phase 4: Quality Assessment
12. **Select risk of bias tool**:
    - RCTs → RoB 2 (Cochrane)
    - Non-randomized → ROBINS-I
    - Diagnostic → QUADAS-2
    - Prognostic → QUIPS
13. **Assess each study** and create summary table/figure

### Phase 5: Synthesis
14. **Qualitative synthesis**: narrative summary grouped by outcome
15. **Quantitative synthesis** (if appropriate):
    - Decide: meta-analysis feasible? (clinical/statistical heterogeneity)
    - If yes → use `/meta-analysis` skill for detailed guidance

### Phase 6: Reporting
16. **PRISMA 2020 checklist**: ensure all 27 items are addressed
17. **Certainty of evidence**: GRADE assessment for each outcome

## Output
Generate a systematic review protocol document with all sections above, ready for team review or PROSPERO registration.
